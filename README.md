# 💬 Mini WhatsApp CRUD App

A mini WhatsApp inspired chat application built using **Node.js, Express.js and MongoDB**.

This project implements complete **CRUD (Create, Read, Update, Delete)** operations for managing chats and messages with a simple and clean user interface.

---

## 🚀 Features

* 💬 Create New Chats
* 📄 View All Messages
* ✏️ Edit / Update Messages
* 🗑️ Delete Chats
* 🔗 REST API Integration
* 🗄️ MongoDB Database Storage
* ⚡ Fast Backend using Express.js
* 📱 Responsive User Interface

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript, EJS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **ODM:** Mongoose

---

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Shubham-web-ai/mini-whatsapp-crud.git
```

2. Navigate to project folder:

```bash
cd mini-whatsapp-crud
```

3. Install dependencies:

```bash
npm install
```

4. Create `.env` file:

```env
MONGO_URL=your_mongodb_connection_string
```

5. Start the server:

```bash
npm start
```

6. Open in browser:

```
http://localhost:8080
```
